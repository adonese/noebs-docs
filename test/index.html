working Key request fields

POST https://beta.soluspay.net/workingKey

Working key is the key used by EBS for card holder's info encryption

Request Body

The request body should be "application/json" encoded, and should contain an object with the following attributes.

ParameterDescription
clientId required
systemTraceAuditNumber required
tranDateTime required
terminalId required
 
var url = new URL("https://beta.soluspay.net/workingKey");
var data = {
    clientId: ...,
    systemTraceAuditNumber: ...,
    tranDateTime: ...,
    terminalId: ...
};
var options = {
    method: "POST",
    body: JSON.stringify(data),
    headers: {
        "content-type": "application/json"
    }
};

fetch(url, options).then(function(response) {
    if (!response.ok) {
        throw new Error(response.statusText);
    }
    console.log(response.json());
})
